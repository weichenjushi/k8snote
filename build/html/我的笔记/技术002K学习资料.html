GitHub 仓库¶

关注 GitHub 仓库可以让你了解最一手的进展，但是信息量一般较大，讨论很多难度也比较大。但对于大乘期的你来讲，这些应该不是问题 。另外，GitHub 仓库还包含很多 Kubernetes 系统内部设计，例如调度器的优化方案、资源垃圾回收方案等，值得了解和学习。
